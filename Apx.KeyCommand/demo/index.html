Apx.KeyCommand Demo

Apx.KeyCommand is a comprehensive keyboard command capturing library. As you may know, capturing key input across browsers and operating systems can get quite nasty. So, KeyCommand takes care of a lot of the hard work for you.

To demo, press any of the following keyboard combinations.

  • SHIFT A
  • SHIFT ALT A
  • F2
  • RETURN/ENTER
